# 🏨 Hotel Reservation System
A simple Java console-based Hotel Reservation System developed using Object-Oriented Programming (OOP). This project allows users to view available rooms, book rooms, cancel bookings, and view booking details.

## 📌 Features
- View Available Rooms
- Book a Room
- Cancel a Booking
- View All Bookings
- Automatic Booking ID Generation
- Payment Simulation
- Console-Based User Interface

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## 📂 Project Structure
HotelReservationSystem/
│── Room.java

│── Hotel.java

│── README.md

## ▶️ How to Run
1. Open the project in IntelliJ IDEA or Eclipse.
2. Compile all Java files.
3. Run `Main.java`.
