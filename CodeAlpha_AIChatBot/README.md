# 🤖 AI ChatBot
A simple Java console-based AI ChatBot developed using Object-Oriented Programming concepts. The chatbot responds to common user questions using predefined responses stored in a HashMap.

## 📌 Features
- Interactive console chatbot
- Rule-based responses
- Case-insensitive input
- Exit command to end conversation
- Uses HashMap for response management

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- HashMap
- Scanner
